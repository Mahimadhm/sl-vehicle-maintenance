# Sri Lankan Vehicle Maintenance Tracking System

## Features
- User Authentication
- Vehicle Management
- Maintenance Logs
- REST API
- Admin Panel
- Tailwind CSS Dashboard
- PostgreSQL Ready
- Render Deployment Ready

## Tech Stack
- Flask
- SQLAlchemy
- SQLite/PostgreSQL
- Tailwind CSS
- Vanilla JavaScript

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

## Deploy to Render
1. Push project to GitHub
2. Create Render Web Service
3. Add DATABASE_URL
4. Deploy

- /api/logs
